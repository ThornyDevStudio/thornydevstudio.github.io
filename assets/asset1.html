Runtime Debugger Toolkit

‚ú® Introduction

This asset consolidates several runtime tools into a single, easy-to-integrate package:

  • Collider Visualizer: Display 2D and 3D colliders in-game.
  • In-Game Console: Show Unity logs in the game window (Editor & Build).
  • FPS Tracker: Real-time FPS and memory tracking via a live graph.
  • Time Scaler: Adjust Time.timeScale for slow-motion or perf tests.

Each tool is accessible via draggable in-game UIs.

üß∞ Installation

Import via Unity Package Manager. Demo requires Cinemachine, Input System & TextMeshPro. For minimal use, drag DebuggerToolManager.prefab into your scene.

Unity version: 2022.3.7f1. Supports URP, Built-in, HDRP (collider tool not HDRP-compatible).

For Built-in pipeline, use the URPToStandardConverter.cs from the menu. The main prefab is DebuggerToolManager, with Tab to unlock/lock mouse (can be changed).

üîß Tool Details

  • Collider Visualizer: Customizable colors. Solid or wireframe. Shows 2D/3D colliders.
  • In-Game Console: Toggle log types, max message count configurable.
  • FPS Tracker: Shows FPS + system info. Dynamic or fixed FPS range.
  • Time Scaler: In-game slider. x2 button boosts upper range. Reset button included.
