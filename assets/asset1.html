Runtime Debugger Toolkit

‚ú® Introduction

This asset consolidates several runtime tools into a single, easy-to-integrate package:

  • Collider Visualizer: Display 2D and 3D colliders in-game.
  • In-Game Console: Show Unity logs in the game window (Editor & Build).
  • FPS Tracker: Real-time FPS and memory tracking via a live graph.
  • Time Scaler: Adjust Time.timeScale for slow-motion or perf tests.

Each tool is accessible via draggable in-game UIs.

üß† Tip

To use wireframe on imported meshes, enable Read/Write in the Model Import Settings.

If you want to render imported meshes (e.g., .fbx, .obj files) in wireframe mode at runtime within a build you must enable Read/Write access on those meshes. Otherwise, Unity will throw the error: "Not allowed to access vertices on mesh..." To enable Read/Write access: 1. Select the mesh asset in the Project window (inside the Assets folder). 2. In the Inspector, navigate to the Model tab. 3. Check the box labeled Read/Write Enabled. 4. Click Apply to save the changes. This ensures Unity allows vertex data access at runtime, which is required for custom rendering such as wireframe overlays.
